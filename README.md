# Numerical Computing & Data Visualization

A collection of Python scripts demonstrating foundational skills in numerical computing with NumPy and data visualization using Matplotlib. This repository serves as a hands-on practice environment for array manipulation, mathematical operations, and creating standard statistical plots.

## Project Structure & Topics Covered

This project is organized into two main focus areas:

### 1. NumPy Operations (`/numpy`)
Scripts demonstrating core mathematical operations and array manipulations:
* Array creation, slicing, and indexing.
* Broadcasting and vectorization.
* Advanced mathematical functions and matrix operations.

### 2. Matplotlib Visualizations (`/matplotlib`)
A comprehensive set of scripts generating various types of data visualizations:
* Line graphs (`1_simplelinegraph.py`)
* Scatter plots (`2_scatterplot.py`)
* Bar charts (`3_barchart.py`)
* Histograms and Pie charts

## Tech Stack Used

* *Python*
* *NumPy* (Numerical computation)
* *Matplotlib* (Data visualization)

## Prerequisites and Installation

To run the scripts in this repository, you will need Python installed along with NumPy and Matplotlib.

```bash
pip install numpy matplotlib
```

## How to Run

You can execute any of the indivitual practice scripts directly from your terminal.
To run a numerical practice task:
```bash
python numpy/task1numpy.py
```

To generate and view a specific plot:
```bash
python matplotlib/ex5_piechart.py
```
(Note: When running visualization scripts, you may need to close the plot window for the script to finish executing.)
